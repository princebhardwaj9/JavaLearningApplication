Java is a object-oriented:

As Java is a object-oriented language so it support fundamental OOP concepts follows:

  • Abstraction
  • Polymorphism
  • Inheritance
  • Encapsulation

So classes & objects is implemented via OOP concepts:

  • Classes: A class is a blueprint or prototype from which objects are created.
  • Objects: An object is a software bundle of related state and behaviour. Example Car has a states- Color, Model as well as behaviours- driving, engine start/stop. An object is an instance of a class. Object is the runtime entity it always get the memory at runtime.

Creating Objects

A class provides the blueprint for objects; you create an object from a class.

Statemet: Demo demo = new Demo();

Above statement has three parts:

  1. Declaration: The word demo is the variable declaration that associate a variable name with an Demo(object) type.
  2. Instantiation: The 'new' keyword is a Java operator that creates the object.
  3. Initialization: The new operator is followed by a call to a constructor, which initializes the new object.
