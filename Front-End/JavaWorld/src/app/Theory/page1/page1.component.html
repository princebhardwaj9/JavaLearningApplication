Introduction to JAVA

Java is a programming language and computing platform first released by Sun Microsystems in 1995.

Java is a high-level object oriented programming language.

There are lots of applications and websites that will not work unless you have Java installed, and more are created every day. Java is fast, secure, and reliable. From laptops to datacenters, game consoles to scientific supercomputers, cell phones to the Internet, Java is almost everywhere.

Learning of language like Javascript, C#, or C++ becomes much easier if you learn java as java derives its syntax from C.




Why We Need java:

As java is open-source & platform independent language. Java used almost everywhere because it is easy to learn. One who want to become good Software Engineer must learn java. You can see the various features of java why you must learn the java.




Features of Java:

Their are many features of Java but the main features of Java are:

  1. Object Oriented: In java everything is represent in the form of object except the primitive data types. Java is based on object model.

  2. Secure: With this feature it enables to develop virus-free systems & it also provides public-key encryption.

  3. Platform Independent: When Java source code is compiled, it is not compiled into platform specific machine, rather into platform independent byte code(.class file). Byte code is distributed over web & interpreted by the Java Virtual Machine(JVM) on whichever platform it is being run on.

  4. Architecture-neutral: Java compiler generates an architecture-neutral object file format, which makes the compiled code executable on many processors, with the presence of Java runtime system.

  5. Portable: It also a primary objective of Java programming language creation was to make it portable. It means that you can run Java bytecode on any hardware that has a compliant JVM.

  6. Multithreaded: This feature allows concurrent execution of two or more parts of a program for maximum utilization of CPU ideal time.

  7. Robust: Java is robust means strong. In Java there is no concept of pointers & also java use strong memory-management & automatic garbage collection.






What Can Java technology Do?

The general-purpose, high-level programming language is a powerful software platform. Every full implementation of the Java platform gives you the following features:

  • Development Tools: The development tools provide everything you'll need for compiling, running, monitoring, debugging and documentating your application. As a new developer, the main tools you'll be using are the javac compiler, the java launcher, and the javadoc documentation tool.

  • Application Prigramming Interface (API): The API provides the core functionality of the Java programming language. It offers a wide array of useful classes for use in your own applications. It spans everything from basic objects, to networking and security, to XML generation and databse access and more.

  • Deployment Technologies: The JDK software provides standard mechaisms such as the Java Web Start Software and Java Plug-In software for deploying your applications to end users.

  • User Interface Toolkits: The JavaFX, Swing and Java 2D toolkits make it possible to create sophisticated Graphical User Interfaces(GUIs).

  • Integration Libraries: Integration libraries such as the Java IDL API, JDBC API, Java Name and Directory Interface (JNDI) API, Java RMI and Java Remote Method Invocation over Internet Inter-ORB Protocol Technology (Java RMI-IIOP Technology) enable database access and manipulation of remote objects.
