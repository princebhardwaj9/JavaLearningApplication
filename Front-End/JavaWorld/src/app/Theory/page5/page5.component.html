Type Conversion?

It is a situation when you assign a value of one data type to another. Both the data types might be compatible or might not be, with each other. If they are compatible the Java performs automatically conversion called as Auto Casting or Widening.

  • Automatic Type Conversion- Also known as widening, Implicit Conversion. See the figure below of widening. It happens when:
    Both the data types are compatible.
    When we assign value of a smaller size data type to a bigger size data type.
  • Explicit Conversion- When we assign value of bigger one to the lesser one data type, then we need to cast it or explicitly convert it.

How to convert?

While evaluating expressions, the intermediate value may exceed the range of operands and hence the expression value will be promoted. Some conditions for type promotion are:


  1. Java automatically promotes each byte, short, or char operand to int when evaluating an expression.
  2. If one operand is a long, float or double the whole expression is promoted to long, float or double respectively.

Examples:

Widening Narrowing
int i = 100;
long l = i; 		long l = 100;
int i = (int)l;
int i = 100;
double d = i; 		double d = 2.1245;
int i = (int)d;
byte b = 125;
short s = b;
int i = s; 		int i = 100000;
short s = (short)i;
byte b = (byte)s;
