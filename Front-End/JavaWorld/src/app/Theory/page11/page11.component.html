    Constructor

  • Constructor is the special member function used to initialize the object.
  • Putting in the values into the properties of an object while creating it this process is called object initialization, or you can say, Putting the values into the data members of an object while creating it.
  • A Java constructor is special method that is called when an object is instantiated. Instantiation means making a object that is using the new keyword.
  • A constructor is used to initialize the state of an object.
  • Constructor always return reference Id.
    • No class can exist without constructor.
  • If their is no constructor in the class then one non-parameterized constructor(Default Constructor) will be inserted into the class by compiler.
  • Simply if you want any task (like database connection, Network) to perform only once in the lifecycle of an object just put the code of that task into the constructor.
  • Whenever you want to initialized the data member of each object with the same values then always use default constructor and this concept is known as static initialization of non-static data member.
  • If you want to initialized the data member of each object with the different different values then always use parameterized constructor and this concept is known as dynamic initialization of non-static data member.
