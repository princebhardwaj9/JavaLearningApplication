Static

It is a special keyword & widely use keyword in Java for memory management. It can be used with variables, methods, blocks and nested classes. Basically, static is used for variable or a method which are same for every object/instance of class.


There are:

  • Static member variable
  • Static member function
  • Static iner class
  • But we don't have static local variable

Static Variables:

  • Static variables are declared in class using static keyword.
  • Static variables are by default initialized to its default value.
  • Static variable has a single copy for the whole class and does not depend on the objects.

Static Function:

  • Static functions defined inside the class are qualified with the keyword static.
  • Static functions can only access static members of same class.
  • Static function can be invoked using class name and dot operator.

    Ex. ClassName.function_Name();


Initialization Block

    There are two types of initialization blocks:

  • Instance Initialization Block
  • Static Initialization Block

Instance Initialization Block:

  • Instance Initialization Block belongs to object.
  • An Initialization Block declared in the class is executed when an instance of class is created.
  • return keyword cannot be used in initialization block.
  • Instance initializers are permitted to refer to the current object via the keyword this and can use the keyword super.
  • It is a block with curly braces & without name.
  • It will be execute when we make object of a class.
  • If you want to make more than one initiallization block it can be possible.
  • Note: Compiler will change all blocks into single block in the same sequence you write the code. After that, compiler will insert all the code of block in the first line of Constructor.

Static Initialization Block:

  • A static initializer declared in a class is executed when the class is initialized.
  • Static initializer may be used to initialize the class variable of the class.
  • return keyword cannot be used in static initialization block.
  • this or super cannot be used in static block.
  • Static initialization block is executed once in life-time of the class regardless how many times you made objects.
  • As same like in instance initialization block compiler will change all static block code into single code.
  • Static initialization block is qualified by the static keyword.
  • Anything you want to do once in lifecycle of the class put that code in static block like database connection, GUI etc.
