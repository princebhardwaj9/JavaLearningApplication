Installation/Environment Setup

As Java is a open-source, so Java SE freely available you can click here to download JDK according to your OS.

After successfully installation you need to set your Environment Variables to set path of JDK.




Follow Below Steps For Setting Up Path In Windows

  1. Click on My Computer then click on Properties


  2. Go to Advance system settings then click on Environment variables.



  3. Change path location to /java/jdk/bin.




Where you edit Java code?

Their are many editors for Java but the most popular are:

  1. Notepad: You can use any text editor like Notepad, TextPad, Notepad++ etc.

  2. Eclipse: It is a Java IDE & also most popular. It is completely a opensource IDE. You can download it from here.

  3. IntelliJ IDEA: This IDE is almost same as eclipse but IntelliJ IDEA has two versios open-source & paid edition also. You can download it from here.

  4. Netbeans: This IDE is also open-source & is available for variety of OS. You can download it from here.
